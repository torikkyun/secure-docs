# TAURI SETUP - LỊCH SỬ THAY ĐỔI

## 1. CÀI ĐẶT DEPENDENCIES
- Cài @tauri-apps/cli @tauri-apps/api
- Cài concurrently (để chạy nhiều lệnh cùng lúc)

## 2. KHỞI TẠO TAURI PROJECT
- Chạy: pnpm tauri init
- Tạo src-tauri folder với Rust code

## 3. THAY ĐỔI PACKAGE.JSON
Scripts thêm:
- "dev:web": "vite dev --port 3005"
- "tauri": "tauri"
- "tauri-dev": "tauri dev"
- "tauri:dev": "tauri dev --config src-tauri/tauri-dev-only.conf.json"
- "dev:both": "concurrently \"pnpm dev:web\" \"pnpm tauri:dev\""
- "tauri-build": "tauri build"

## 4. THAY ĐỔI .ENV
- NODE_ENV: production → development (yêu cầu của Vite dev)

## 5. THAY ĐỔI TAURI CONFIG (tauri.conf.json)
- devUrl: "http://localhost:3000"
- beforeDevCommand: "pnpm dev:web"
- Window settings:
  * fullscreen: false (chỉ dùng maximized thay vì fullscreen)
  * decorations: true (hiển thị thanh tiêu đề + nút close/minimize/maximize)
  * maximized: true (mở maximized lúc khởi động)
- CSP: cho phép ws/wss trên localhost:3000
- Loại bỏ devTools property (không hợp lệ)

## 6. TẠO TAURI DEV-ONLY CONFIG (tauri-dev-only.conf.json)
- Giống tauri.conf.json nhưng:
  * Không có beforeDevCommand (dùng khi web server đã chạy)
  * devUrl: "http://localhost:3000"

## 7. THAY ĐỔI REACT ROOT COMPONENT (__root.tsx)
- Thêm F12 blocker (KeyboardEvent handler):
  * F12
  * Ctrl+Shift+I (DevTools)
  * Ctrl+Shift+C (Inspect Element)
  * Ctrl+Shift+J (Console)
  * Ctrl+Shift+K (Console)
- Chỉ render TanStackDevtools khi NODE_ENV=development

## 8. TỐI ƯU VITE CONFIG (vite.config.ts)
- Tắt devtools() plugin khi không phải development mode
- Thêm build optimization:
  * Code splitting: React, Router, Query riêng biệt
  * Terser minify bật
  * Xóa console và debugger statements
  * Filter plugins (loại bỏ undefined)

## 9. TỐI ƯU RUST BUILD (.cargo/config.toml)
- Parallel compilation: 4 jobs
- LTO (Link-Time Optimization) bật
- Codegen units: 1
- Strip symbols (giảm file size)

## CÁCH CHẠY

### Development (Web + Tauri cùng lúc):
```
pnpm dev:both
```

### Development (Web riêng):
```
pnpm dev:web
```

### Development (Tauri riêng - nếu web đã chạy):
```
pnpm tauri-dev-only
```

### Development (Tauri tự khởi động web):
```
pnpm tauri-dev
```

### Build Production:
```
pnpm tauri-build
```

## NHỮNG GÌ ĐÃ ĐƯỢC TÍNH ĐẾN

✅ F12 bị chặn hoàn toàn
✅ App maximized lúc khởi động
✅ Thanh decorations (tiêu đề + nút close/min/max)
✅ Port 3005 cho web server
✅ CSP cho phép backend trên localhost:3001
✅ DevTools chỉ load khi development
✅ Code splitting + minify cho production
✅ Rust optimization cho build nhanh hơn
✅ TanStack devtools chỉ show khi dev mode

## PORT CONFIGURATION

- Web Server: 3000
- Backend API: 3001
- DevTools EventBus: 42069 (Tauri internal)

## ENVIRONMENT

- NODE_ENV: development (cho vite dev)
- SESSION_SECRET: (từ .env file)
- VITE_BACKEND_URL: http://localhost:3001/api
